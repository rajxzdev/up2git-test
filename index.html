Panduan & Tutorial up2git

Langkah mudah menggunakan up2git untuk upload file ZIP ke GitHub lewat HP.

1
Cara Membuat GitHub Personal Access Token (PAT)

Untuk menghubungkan up2git dengan akun GitHub Anda, Anda membutuhkan token khusus:

  • Buka browser HP Anda lalu login ke github.com.
  • Klik foto profil Anda di pojok kanan atas, lalu pilih Settings.
  • Scroll ke paling bawah di menu sebelah kiri, klik Developer settings.
  • Pilih menu Personal access tokens > Tokens (classic).
  • Klik tombol Generate new token (classic).
  • Beri nama pada note (misal: up2git HP).
  • Pada bagian Select scopes, centang kotak repo (Full control of private repositories) dan read:user.
  • Scroll ke bawah lalu klik tombol Generate token.
  • PENTING: Salin (copy) kode token yang muncul dan simpan. Kode ini hanya ditampilkan satu kali!
2
Cara Connect ke up2git

Buka menu Home di aplikasi ini, paste kode token yang sudah Anda salin ke dalam kolom input token, lalu klik tombol Connect to GitHub. Jika berhasil, foto profil dan statistik GitHub Anda akan langsung muncul.

3
Cara Memilih Repository Tujuan

Anda bisa memilih repository lewat 2 cara:

  • Buka menu Repositories di bawah layar, cari repo yang Anda inginkan, lalu klik tombol Upload ke sini.
  • Atau langsung buka menu Upload dan pilih nama repository dari menu dropdown di atas.
  • Jika belum punya repo, buka menu New Repo untuk membuat repository baru secara praktis.
4
Cara Upload File ZIP beserta Strukturnya

Kompres folder project Anda di HP menjadi format .zip:

  • Di menu Upload, klik kotak area Drag & Drop lalu pilih file .zip Anda.
  • up2git akan mengekstrak seluruh file beserta nested foldernya secara otomatis langsung di browser HP.
  • Pratinjau seluruh file dan ukurannya akan ditampilkan di layar.
  • Klik Upload Seluruh Isi ke GitHub, lalu konfirmasi.
  • Tunggu progress bar sampai 100%. File Anda sudah online di GitHub!
